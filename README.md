# Spinningfish

Spinningfish는 카메라 기반 시선 유지 게임입니다. `WATCH FISH`를 클릭하고 물고기를 계속 바라본 뒤, 시선이 이탈하면 기록을 로컬 랭킹에 저장합니다.

## 로컬 실행

저장소 루트에서 정적 서버를 실행합니다.

```sh
python3 -m http.server 4173 --bind 127.0.0.1
```

그다음 아래 주소를 엽니다.

```text
http://127.0.0.1:4173
```

게임 플레이에는 카메라 권한이 필요합니다. 브라우저 카메라 권한은 보통 `localhost`, `127.0.0.1`, HTTPS 환경에서 동작합니다.
